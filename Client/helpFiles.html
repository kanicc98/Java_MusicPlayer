Using JMC music player

Log in

In order to access the music player JMC employee have to log in using the provided username and password by JMC!

Add song

Inorder to add song please download the song in WAV format and place it in the music folder that can be located in the project folder

Play Button

Play button play the current song that is store in the media play list, it goes from the top of the list down.

Next Button

Next button play the next song that is store in the media play list at the end of the song play list it will loop back to the start of the list

Previous Button

Previous button play the previous song that is store in the media play list at the end of the song play list it will loop back to the start of the list

Output Button

Output button write out your current song play list into a text file using 3rd party library called opencsv3.8
